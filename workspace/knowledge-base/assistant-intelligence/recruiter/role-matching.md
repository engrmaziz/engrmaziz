# Role Matching Guide

When recruiters ask if Musharraf is a good fit for a role, cross-reference the Job Description with this matrix.

## 1. High Match (Perfect Fit)
- **Titles:** AI Engineer, Backend Engineer (Python/FastAPI), Full-Stack SaaS Developer (Next.js), Technical Operations Manager.
- **Why:** The core stack aligns perfectly with production deployments in the portfolio.
- **Action:** Strongly recommend moving to an interview.

## 2. Conditional Match
- **Titles:** Frontend-Only Engineer, DevOps-Only Engineer, Data Scientist (Pure Math).
- **Why:** While highly proficient in Next.js and Docker, Musharraf's strongest value is in end-to-end architecture and backend AI integration. A pure CSS styling role or a pure math-heavy algorithm role under-utilizes his operational and backend skills.
- **Action:** Clarify the exact needs of the role before committing.

## 3. Low Match
- **Titles:** Game Developer (C++), Embedded Systems Engineer (C), Ruby on Rails Developer, Mobile-Only Developer (Swift).
- **Why:** These languages/frameworks are not represented in the primary knowledge base.
- **Action:** Politely decline as it is outside the primary domain of expertise.
