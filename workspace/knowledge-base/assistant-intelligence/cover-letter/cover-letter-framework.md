# Cover Letter Framework

This framework allows the AI Assistant to generate highly targeted, deterministic cover letters based *only* on the factual knowledge base.

## 1. The Hook (Paragraph 1)
- **Objective:** Grab attention by mapping a specific requirement in the Job Description to a high-impact metric.
- **Logic:** If Job requires [Scaling], mention [NovaSole 500k visitors]. If Job requires [AI Security], mention [LLM Guardrail / AIHK zero hallucinations].

## 2. The Evidence (Paragraph 2)
- **Objective:** Prove the Hook using a specific architectural implementation.
- **Logic:** Detail the exact stack (e.g., FastAPI, Next.js, Qdrant) used to achieve the metric. Emphasize *ownership* (e.g., "I architected...", "I managed a team of 14...").

## 3. The Business Value (Paragraph 3)
- **Objective:** Connect the technical implementation back to business ROI.
- **Logic:** Use the `achievement-library.md` to highlight fault reduction (18%), SLA enforcement, or compliance (HIPAA).

## 4. The Call to Action (Conclusion)
- **Objective:** State enthusiasm for the specific company and request an interview.
- **Format:** Keep it brief, professional, and confident. Do not sound desperate.
