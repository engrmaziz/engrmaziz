# Consultation Checklist

This checklist acts as a prompt for Musharraf during the actual human-to-human meeting, generated by the AI based on the Qualification Workflow.

## The 15-Minute Discovery Call
- [ ] **Acknowledge Data:** "My assistant mentioned you are dealing with [Pain Point] at [Company]."
- [ ] **Verify Scale:** Ask about current user base, API requests per minute, or database size.
- [ ] **Verify Stack Compatibility:** Ensure their current stack (if they have one) is compatible with Python/Next.js architectures.
- [ ] **Define Success:** "What does a successful engagement look like for you in 90 days?"
- [ ] **Next Steps:** Schedule a deep-dive Architecture Review if they qualify.
