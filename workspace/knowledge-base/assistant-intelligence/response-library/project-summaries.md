## 1. SHBRAG (Self-Healing RAG)
"I built SHBRAG, a Corrective RAG pipeline using LangGraph and Qdrant. It evaluates its own retrieved context before generating an answer. If the data is poor, it falls back to a web search, ensuring the AI never hallucinates. This architecture was crucial for my clinical AI deployment."

## 2. AegisFlow (FinTech SaaS)
"AegisFlow is a full-stack FinTech application I built using Next.js and FastAPI. It demonstrates my ability to handle complex relational data in PostgreSQL, secure API routes, and deploy a modern, scalable web architecture."

## 3. LLM Guardrail Gateway
"To ensure data privacy when using LLMs, I built a Guardrail Gateway using FastAPI and Microsoft Presidio. It intercepts all prompts and redacts 18+ types of PII (like names and credit cards) before sending the payload to OpenAI, ensuring absolute HIPAA/PCI compliance."

## 4. VoiceRAG Core v1
"I developed VoiceRAG Core to achieve ultra-low latency conversational AI. By replacing standard REST APIs with WebSockets and utilizing Groq LPUs, I achieved sub-500ms voice response times, allowing for natural human-AI conversations with barge-in support."
