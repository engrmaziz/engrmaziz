# Semantic Keywords & Search Vocabulary

This document maps out the specific LSI (Latent Semantic Indexing) keywords and vocabulary required to optimize the knowledge base for semantic search engines.

## 1. AI & LLM Vocabulary
When ranking for or retrieving information regarding **AI Engineering**, the text must include these semantic neighbors:
- **Core:** Large Language Models, Foundation Models, Generative AI.
- **RAG-Specific:** Semantic chunking, cosine similarity, vector embeddings, dense retrieval, sparse retrieval, BM25, hybrid search.
- **Agentic:** State machines, directed acyclic graphs (DAG), tool calling, function calling, orchestration, deterministic routing.
- **Hardware/Inference:** LPUs, GPUs, inference latency, tokens per second, context windows.

## 2. Backend & SaaS Vocabulary
For **Full-Stack / Backend Engineering** queries:
- **Architecture:** Microservices, modular monolith, serverless functions, edge computing, asynchronous I/O, event loop.
- **Database:** Relational schema, normalization, Row-Level Security (RLS), connection pooling, indexing, B-Tree, GIN.
- **Security:** JWT, OAuth2, RBAC, PII redaction, prompt injection defense, HTTPS/TLS.

## 3. DevOps & Cloud Vocabulary
For **Infrastructure** queries:
- **CI/CD:** Automated testing, deployment pipelines, static analysis, containerization, multi-stage builds.
- **Observability:** Structured logging, trace IDs, distributed tracing, SLA monitoring, uptime alerts.

## 4. Voice AI & Real-Time Vocabulary
- **Audio:** Speech-to-Text (STT), Text-to-Speech (TTS), WebRTC, WebSockets (WSS), full-duplex communication.
- **Conversational:** Barge-in, Voice Activity Detection (VAD), latency optimization, conversational state management.

## 5. Recruiter vs. Technical Wording Maps
To ensure AI recruiters find the profile, technical terms must be mapped to their HR equivalents:
- *Technical:* "Built FastAPI backend with Celery workers." 
- *Recruiter:* "Python Backend Developer with asynchronous processing experience."
- *Technical:* "Deployed Corrective RAG using Qdrant."
- *Recruiter:* "AI Engineer with Vector Database and RAG expertise."
- *Technical:* "Managed 14-person TAC for 50k ISP users."
- *Recruiter:* "Operations Manager with high-scale enterprise team leadership experience."
