# Frontend Development & UI/UX

## Definition
The engineering of client-side web applications, focusing on delivering exceptional user experiences through highly performant, accessible, and visually striking interfaces. This involves deep knowledge of browser rendering cycles, state management, and modern CSS paradigms.

## Core Technologies & Frameworks

### 1. Next.js & React
- **Definition:** Next.js is a React framework that enables server-side rendering (SSR), static site generation (SSG), and React Server Components (RSC).
- **Experience Level:** Expert
- **Projects:** AuraNode, AegisFlow, dentl2, Green Navigator, OpenScholar.
- **Specialization:** App Router architecture, Turbopack, data fetching strategies, hydration optimization.

### 2. TypeScript & JavaScript
- **Definition:** The foundational languages of the modern web. TypeScript adds static typing to JavaScript.
- **Experience Level:** Expert
- **Specialization:** Strict type checking, generics, complex interfaces for API payloads.

### 3. Styling & CSS
- **Technologies:** Tailwind CSS (v4), Vanilla CSS, Framer Motion (for animations).
- **Experience Level:** Expert
- **Specialization:** Utility-first CSS, responsive design, dark mode implementation, complex CSS grid/flexbox layouts.

### 4. 3D Graphics & WebGL
- **Technologies:** Three.js.
- **Experience Level:** Intermediate to Advanced
- **Projects:** dentl2.
- **Specialization:** Rendering interactive 3D models in the browser, memory management (preventing context leaks).

### 5. State Management
- **Technologies:** Zustand, React Context, Redux (familiar).
- **Experience Level:** Expert
- **Specialization:** Passing dehydrated state from Server Components to Client Components efficiently.

## Related Interview Questions
- "When would you choose a React Server Component over a Client Component in Next.js?"
- "How do you manage complex, highly interactive state (like a shopping cart) in Next.js without causing hydration errors?"
- "Explain your approach to optimizing Core Web Vitals for an e-commerce platform."

## Future Growth
Deepening knowledge in WebAssembly (Wasm) to port heavier backend processes directly to the client browser.
