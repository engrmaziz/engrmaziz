# AI & Machine Learning Engineering

## Definition
The application of artificial intelligence, specifically Large Language Models (LLMs) and Vector Mathematics, to solve complex enterprise problems. This encompasses everything from basic prompt engineering to complex, cyclic multi-agent orchestration and deterministic output enforcement.

## Core Technologies & Frameworks

### 1. LangGraph
- **Definition:** A library for building stateful, multi-actor applications with LLMs.
- **Experience Level:** Expert
- **Projects:** VoiceRAG Core v1
- **Industry Demand:** Very High (Standard for complex AI agents)

### 2. Retrieval-Augmented Generation (RAG)
- **Definition:** Architecture that augments LLMs with external, private knowledge bases.
- **Experience Level:** Expert
- **Specialization:** Corrective RAG (CRAG), Semantic Chunking, Hybrid Retrieval.
- **Projects:** Self-Healing RAG Pipeline (SHBRAG)

### 3. Vector Databases
- **Technologies:** Qdrant, ChromaDB, Pinecone, Supabase Vector, Cloudflare Vectorize.
- **Experience Level:** Advanced
- **Specialization:** High-dimensional vector indexing, metadata filtering, cosine similarity search.

### 4. Model Context Protocol (MCP)
- **Definition:** Open standard for connecting AI models to external tools and data sources.
- **Experience Level:** Advanced
- **Projects:** Git Archaeologist MCP Server

### 5. Traditional ML & Data Science
- **Technologies:** PyTorch, scikit-learn, Pandas, NumPy.
- **Experience Level:** Intermediate to Advanced
- **Projects:** AegisFlow (Fraud Detection)
