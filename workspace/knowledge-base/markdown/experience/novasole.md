# Software Engineer & IT Manager

## Overview
A heavy full-stack engineering role focused on architecting, scaling, and maintaining a high-traffic e-commerce platform. This role demanded deep expertise in modern web frameworks (Next.js), database optimization, and third-party API integrations (payments, shipping).

## Company
**NovaSole Pakistan**

## Industry
E-Commerce / Retail Tech

## Employment Type
Full-Time

## Dates
**December 2023 – August 2024**

## Location
Kasur, Pakistan

## Responsibilities
- Architect and develop the core e-commerce storefront and admin dashboard using Next.js and TypeScript.
- Design and maintain relational database schemas (PostgreSQL/MySQL) for product catalogs, user data, and complex order routing.
- Integrate critical third-party REST APIs, including payment gateways and logistics providers.
- Manage IT operations, ensuring high availability during traffic spikes and maintaining data integrity across 3 distinct sales channels.

## Technical Stack
- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Next.js API Routes (Serverless)
- **Databases:** PostgreSQL, MySQL
- **Integrations:** Payment Processors, REST APIs

## Architecture & Systems Worked On
1. **Core E-Commerce Engine:** A highly performant, SEO-optimized frontend built to handle massive concurrent traffic without degrading Core Web Vitals.
2. **Data Synchronization Pipeline:** A backend service ensuring inventory and order status remain perfectly synced across three separate sales channels in real-time.

## Business Problems & Solutions
- **Problem:** E-commerce platforms often suffer from data fragmentation across different sales channels, leading to overselling or lost orders.
  - **Solution:** Engineered a robust, centralized data pipeline achieving 98%+ data accuracy across all channels.
- **Problem:** High traffic spikes during sales events causing database locks and slow page loads.
  - **Solution:** Leveraged Next.js static generation (SSG) for product pages and optimized PostgreSQL queries to handle 500k+ visitors seamlessly, resulting in average monthly sales of 10M PKR.

## Achievements & KPIs
- Successfully scaled the platform infrastructure to reliably handle **500,000+ monthly visitors**.
- Maintained a pristine **98%+ data accuracy** rate across complex, multi-channel inventory systems.
- Served as the inspiration and technical testing ground for the highly advanced `AuraNode` (Organic Harvest) project.

## Business Impact
Directly drove revenue by ensuring the platform remained lightning-fast and highly available during peak traffic. The API integrations streamlined the checkout process, reducing cart abandonment and eliminating manual logistics overhead.

## Recruiter Highlights
- Proven ability to handle extreme scale (500k+ users) on modern stacks (Next.js).
- Strong understanding of commercial/financial software logic (e-commerce, payments).
- Dual capability in both frontend aesthetics and heavy backend data engineering.

## Interview Questions
- "How did you manage state and caching in Next.js to handle 500,000 monthly visitors?"
- "Explain the architecture of your data synchronization pipeline. How did you ensure 98%+ accuracy across 3 sales channels?"
