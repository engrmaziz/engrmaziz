# AI Fluency: Framework & Foundations

## Overview
A specialized certification focusing on interacting with advanced Foundation Models. It emphasizes structured prompting, in-context learning, and utilizing AI as a determinable software component rather than a generic chatbot.

## Provider
**Anthropic / UCC**

## Skills & Learning Outcomes
- Advanced Prompt Engineering (Chain of Thought, Few-Shot prompting).
- System Prompt design for enforcing AI personas and output schemas.
- Understanding context windows and token optimization.

## Industry Value
Anthropic is known for its focus on AI safety and "Constitutional AI". This certification proves an ability to design prompts that yield safe, predictable, and highly structured outputs (JSON/XML) suitable for API consumption.

## Projects Using It
Directly applied in the `Git Archaeologist MCP Server` to design the system prompts that allow Claude to parse local git histories without hallucinating missing commits.
